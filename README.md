# 🛍️ Mall Customer Segmentation using K-Means

## 📌 Project Overview
This project uses K-Means clustering to segment mall customers based on their income and spending score.

## 📊 Dataset
- Mall Customers dataset
- Features: Age, Income, Spending Score

## ⚙️ Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## 🚀 How to Run
```bash
pip install -r requirements.txt
python app.py
